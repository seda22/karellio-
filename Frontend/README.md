# Proje Kurulumu

### Gereksinimler

Node.js (v18+ önerilir)

npm

### Kurulum
```
git clone https://github.com/YOUR_USERNAME/karellio-.git

cd karellio-

npm install
```
### Çalıştırma
```
npm run dev
```
### Kullanılan Teknolojiler

React

Vite

MUI Joy UI (@mui/joy)

JavaScript (ES6)

CSS

### Teknik Detaylar

State Yönetimi: React useState

Bileşen Bazlı Mimari

UI Sistemi: MUI Joy Grid, Card, Typography
